# Guides

Task-oriented walkthroughs.

- [using-the-cms](./using-the-cms.md) is how to run the admin: signing in,
  editing content, the asset library and galleries, layout, moderation, and
  previewing.
- [extending](./extending.md) is how to grow the site in code: add a data
  source, add a module, or add and split a nav node. Each is a small, local
  change by design.
