# Reference

Lookup material. These track the code closely, so each one names the file that
owns the real values and describes shapes rather than pasting snapshots that go
stale.

- [http-api](./http-api.md) is every HTTP endpoint: the read API, the public
  write endpoints, presence, assets, the CMS API, and auth, with status codes.
- [configuration](./configuration.md) is every environment variable, its
  default, and what turns each optional feature on.
- [commands](./commands.md) is the pnpm scripts and the two CLIs (sync and
  analytics ingest).
